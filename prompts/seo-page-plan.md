# SEO Page Plan Prompt

Use this prompt only with stored query, autocomplete, SERP, evidence, score, and
competitor records.

Output one of these decisions:

- `build_now`
- `build_after_more_evidence`
- `do_not_build_seo_too_weak`

Every SEO plan must include:

- Keyword clusters.
- Page types.
- Target query.
- User intent.
- Proposed title.
- H1.
- CTA.
- Evidence source.
- Priority.
- Why each page should or should not be built.
- Keyword-to-page map.
- Internal linking plan.
- Prioritized build order.

Required page types:

- Money page.
- How-to guide.
- Comparison page.
- Problem/troubleshooting page.
- FAQ/trust page.

Rules:

- Do not recommend generic blog posts.
- Do not invent search volume.
- Do not treat SEO interest as proof of willingness to pay.
- Tie every page recommendation to stored evidence.
